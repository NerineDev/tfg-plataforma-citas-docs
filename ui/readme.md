# UI

This folder contains visual material related to the user interface of the project.

Its main purpose is to store screenshots and interface references used in the public documentation, especially those that reflect the final implemented MVP more accurately than early planning sketches.

Contents may include:

- Real application screens
- Interface captures used in PI_E3
- Final UI references for documentation
- Selected visual material supporting navigation or feature explanation

This folder is not limited to early wireframes.  
When relevant, implemented screens are prioritized because they represent the final structure, navigation, and behavior of the application more faithfully.

# UI

Esta carpeta contiene material visual relacionado con la interfaz de usuario del proyecto.

Su finalidad principal es reunir capturas e imágenes de la interfaz utilizadas en la documentación pública, especialmente aquellas que reflejan con mayor fidelidad el MVP implementado frente a los bocetos iniciales.

Aquí pueden incluirse:

- Pantallas reales de la aplicación
- Capturas utilizadas en PI_E3
- Referencias finales de interfaz para la documentación
- Material visual de apoyo para explicar navegación o funcionalidades

Esta carpeta no se limita a wireframes iniciales.  
Cuando resulta más útil para la documentación, se priorizan capturas del sistema implementado, ya que representan con mayor precisión la estructura, navegación y comportamiento definitivo de la aplicación.
